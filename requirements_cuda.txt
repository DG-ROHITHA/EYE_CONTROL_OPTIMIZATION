# GPU-SPECIFIC NOTES

# CUDA Installation:
# 1. Download NVIDIA CUDA Toolkit: https://developer.nvidia.com/cuda-downloads
# 2. Download cuDNN: https://developer.nvidia.com/cudnn
# 3. Add CUDA to PATH (Windows: C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.x\bin)

# CuPy Installation:
# - For CUDA 11.x: pip install cupy-cuda11x
# - For CUDA 12.x: pip install cupy-cuda12x
# - Test: python -c "import cupy; print(cupy.cuda.runtime.getDeviceCount())"

# OpenCV with CUDA:
# - Pre-built wheels usually have CUDA disabled
# - Build from source: https://github.com/opencv/opencv-python
#   With cmake flag: -D WITH_CUDA=ON -D CUDA_TOOLKIT_ROOT_DIR=/path/to/cuda

# ============================================
# TROUBLESHOOTING
# ============================================

# CuPy import fails:
#   - Verify CUDA installation: nvcc --version
#   - Check PATH includes CUDA bin directory
#   - Try: python -c "import cupy; cupy.cuda.Device()"

# OpenCV CUDA modules not found:
#   - Check: cv2.cuda.getCudaEnabledDeviceCount()
#   - May need to rebuild OpenCV from source with CUDA support

# MediaPipe GPU delegate fails:
#   - Fallback to CPU automatically (no action needed)
#   - Check: python -c "import mediapipe; print(mediapipe.__version__)"

# PyAutoGUI requires admin privileges on some systems:
#   - Run with elevated permissions or use simulation mode
